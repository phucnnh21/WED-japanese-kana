  HIRAGANA SCRIPT


Hiragana (ひらがな) - Soft script, is one component of Japanese Writing System.

Hiragana is used to write various grammatical and function words, suffixes following a Kanji root and so on...



This is a "simple" kana, which is the kana to start when learning Japanese language. This script is used in Japanese learning books for beginner and in books for children. So that, learning this kana is VERY IMPORTANT!!!
