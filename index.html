  JAPANESE KANA INTRODUCTION


Japanese writing system consists of 3 script:
  • Hiragana (ひらがな)
  • Katakana (カタカナ)
  • Kanji (漢字)

Here is an example sentence:
私 は パーク です。

Romaji: Watashi wa Park desu.

(Meaning: I'm Park.)

Mastering these script is extremely important for people who want to learn Japanese. So that, I decide to create this website to introduce Japanese script and how to write them!
You can access pages in navigation bar to find out each script!
